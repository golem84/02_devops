# 02_devops
Данный репозиторий демонстрирует навыки работы с git и github.
Выполнена генерация ssh-ключа/проверка его работы на другом репозитории.

Репозиторий создан в локальном каталоге командой `git init`
Добавлен файл README.md, в файл внесена запись "# 02_devops"

Выполнен `git add README.md`
Выполнен `git commit -m "init repo"`
Выполнен `git remote add origin https://github.com/golem84/02_devops.git`
Выполнен `git push -u origin master`

Команда `git status` отображает текущее состояние репозитория.

Создана новая ветка от master `git checkout -b feat/#001` и автоматическое переключение на эту ветку.
Разницу между ветками можно посмотреть командой `git diff master feat/#001`

Для просмотра изменений по сравнению с master используем `git diff`
В файл README.md внесены изменения, файл README.md добавлен в индекс `git add README.md`
Изменения сохранены `git commit -m "update README.md"`

Подготовка к слиянию: перейти в ту ветку, куда будем сливать изменения: `git checkout master`
Смотрим изменения между ветками `git diff feat/#001`

Создана новая ветка от master `git checkout -b feat/#002` и автоматическое переключение на эту ветку.
Подготовка к разрешению конфликтов. В данной ветке изменена строка 22.

Для предварительного просмотра изменений с удаленной веткой выполнить `git fetch`
Для вытягивания изменений и применения к отслеживаемым tracked веткам выполнить `git pull`

Для отправки изменений и всех веток в удаленный репозиторий выполнить `git push origin --all`
